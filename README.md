# EmailApp

This is a java project, which generates for new employees an email address and a random password,
after the generation of the random password, the employee has the possibility to change his password
 with one he wants. The email address is composed as follows: the name in capital letters,
 a period, the first name in small letters, the name of the department, the name of the company and at the end ".com".
 After the code is run, an info will be displayed at the end with the password, full name, email address and mailbox capacity
